Redis 数据操作

2018.01.14 Kyrene Wang Tech  热度 ℃
  • redis是key-value的数据，所以每个数据都是一个键值对
  • 键的类型是字符串
  • 值的类型分为五种：
  1. 字符串string
  2. 哈希hash
  3. 列表list
  4. 集合set
  5. 有序集合zset
  6. 数据操作的全部命令，可以查看中文网站
