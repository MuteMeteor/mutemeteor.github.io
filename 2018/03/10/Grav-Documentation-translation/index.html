Grav 文档目录翻译

发表于 2018-03-10 | 分类于 Tech

对于技术文档工程师来说，中英互译能力必不可缺，一个文档工程师一定也是优秀工程师文化的传播者。技术文档的类型多为说明文，要求语言翻译精炼、准确、不拖泥带水，讲究用最少的话讲最明白的道理。所谓实践出真知，为了能更深入的体验技术文档的翻译过程，我尝试寻找优秀的技术文档去实践翻译，在GitHub上fork了Beautiful Docs并从中挑选了一篇文档——Grav Documentation。由于时间有限只对文档目录进行了翻译，希望有机会能完成整篇文档的翻译。

Grav是一个开源的文本文件内容管理系统。Grav 和 Github 充分结合，使用 Markdown 解析文章，具有快速检索和高亮的功能。
