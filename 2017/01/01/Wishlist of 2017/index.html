Plan List of 2017

发表于 2017-01-01 | 分类于 子不语


2017计划清单

思想是具有磁性的，并且有着某种频率。当你思考时，那些思想就发送到宇宙中，然后吸引所有相同频率的同类事物。所有发出的思想，都会回到源头——你。﻿

First Goal

  • 12.23/4 考研PKU CS学硕 380+
    政治65 英语75 数学120 专业课120

英语

  • 认真上本专业的课，尤其英汉翻译和口译基础
  • 恋恋有词U1-U40
  • 整理做过的阅读里的词组和单词到excel表格，定期打印pdf
  • 1997-2017年20年的所有考研阅读做3遍，弄清长难句，吃透真题出题题点
  • 翻译至少20篇计算机相关的paper或网站文章

数学

  • 参加1月份的美赛，自学Matlab和LaTex，并努力和队友拿M奖
  • 从3月份开始，保证每天做数学题至少3h，一周至少五天
  • 一轮 3-4月 《超级通俗》高数，李永乐线代，曹显兵概率论
  • 二轮 5-6月 全书复习（高数部分），吃透每一道题
  • 强化 7-8月 660题+1000题一遍
  • 真题 9-10月 真题30年
  • 模拟 11-12月 张四、张八、合工大、李王

专业课(907)

  • 数据结构：天勤，王道，北大数据结构学习指导
  • 计算机组成原理：慕课，《计算机组成与设计》，橙皮儿书，王道选择题
  • 操作系统：慕课，李善平，王道
  • 计算机网络：谢希仁，王道
  • 408真题，王道田勤模拟

上机+项目

  • 基础：linux操作系统和MySQL数据库
  • 以python为主，每天至少敲实验楼一小节的代码。至少做两个大项目
  • 暑假，每天4h+上机，9月PAT考试甲级80+分。

读书和paper

  • 每天读书1小时，一周至少坚持4天
  • 17年读至少50本书，每一本读过的书或写读书笔记或做mind map
